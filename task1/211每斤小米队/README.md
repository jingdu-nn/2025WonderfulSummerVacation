1. **曹瀚文** 作为项目负责人，负责整体架构和核心功能

3. **刘子轶** 负责测试

## 技术栈

项目包含多个Python文件，涵盖了不同的功能模块：
- `1.py` - 核心功能模块
- `2.py` - 辅助功能模块  
- `all (2).py` - 集成功能
- `all_l.py` - 左侧功能模块
- `all_r.py` - 右侧功能模块
- `vision_s_curve_runner.py` - 视觉S曲线运行器
